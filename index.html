Project Idea

I want to create a Role-playing game (RPG) as this is a large genre in the gaming industry. My game will be focused on forming character bonds, that when certain actions occur it can make the user feel real emotion as they feel like they are apart of the game. The main focus of the game will be creating multiple choices and multiple ends as seen in games such as ‘Until Dawn’. This project will be a choice-based game that if one action occurs then a multitude of events will unfold due to the players choice. However, unlike games such as Until Dawn it will have fighting aspects from games such as ‘Mortal Kombat ‘and combine these two aspects to form a new genre of game.

The reason for this project being interesting is that it has never been done before, combining a choice game with such intense fight mechanics. Along with it never being done before the sales of these two example games are high, ‘Until Dawn was the second best-selling software for the week of 29 August 2015, debuting at No. 2 in the UK retail software sales chart’ and the Mortal Kombat franchise has sold over 32.5 million copies. Not only did these two games do well in sales they also received very high reviews from customers and critics. Along with both these games doing well the other motivation for creating this project is that the game industry is projected to be worth 180.1 billion is revenue by the year 2021. However, I wish to see the gaming industry become even bigger than that and therefore I'm interested in creating a game such as this too bring in new customers and grow the industry further. The reason I want this to happen is I am passionate about gaming I believe that it brings people together and is an outlet for anger and stress leading to a happier and healthier life.

Links for the statistical evidence - https://vgsales.fandom.com/wiki/Best_selling_game_franchises https://en.wikipedia.org/wiki/Until_Dawn https://venturebeat.com/2018/04/30/newzoo-global-games-expected-to-hit-180-1-billion-in-revenues-2021/

The features of the project will include the interactions of user and NPC, a story will be told however the story will vary based on user choice and decision. A user decision can lead to a resolution keeping them from fighting or lead too conflict based on the user's attributes in different skills that they choose to learn. The game will also force users to look around for items and scavenge supplies in order to keep their characters alive, this means the user will have to learn the proper preparations to ration their supplies. Some areas of the game will force the user to use flashlights in order to see, the flashlights will need batteries and these batteries will need to be rationed because if the darkness engulfs them, their character will begin to go insane. When a character reaches insanity, this will cause 1 of 3 actions to occur, the first being he kills all other characters in the party leaving only one alive, the second he runs of and dies in the darkness leaving you with your remaining 3 characters, or the 3rd option he develops a tolerance too the insanity and presses on in the dark. However, this 3rd option will only have a 1% chance of occurring the 2nd will have an 89% chance and the first will have a 10% chance. This gives a certain chance to each part of the game and having skills in luck will increase the 3rd options chance, as an incentive for the user to choose luck as their skill. Similar occurrences will happen when characters begin to starve, if a character starve again 3 options will occur. The first option is the character eats one of his party members causing them to die and thus leaving the user with only 3 characters left along with a permanent status effect that the character is now a cannibal and only can eat other people, meaning the user will have to pick fights and harvest the dead bodies in order to keep this character alive. The second option is the character dies from starvation. The 3rd option is that the character drinks any available water in the players inventory which restores the character back too full hunger and also gives the character a permanent buff which enables the character to continue to consume water to gain hunger and thirst. Again, these will all have certain percentage chances and will increase based on the luck skill of the character. But the base chances for option 1 are 10% chance the 2nd a 89% chance and the 3rd a 1% chance. In addition, to these game mechanics the game will be very unforgiving whatever choice you make or whatever occurs in the game is irreversible and you must play the rest of the game with whatever happens or restart from scratch. This game is all about survival and making the right choices along with item management. The fighting aspect of the game occur when you anger an NPC by saying the wrong thing, if your character is too die in this battle then they are permanently dead. However, if you are too emerge victorious then the user takes all the NPC’s items however they all will have special move sets that a player must learn fast and the better the items the harder the difficulty, however it become easier based on the skill strength which a user can decide to choose over luck. By creating this game, it will open a new market of a fighting style RPG game that has not only choices but strategic rationing and fight mechanics. This game is meant to force the user to learn from their mistakes and be one of the hardest games mentally and skill wise bringing together strategy and combat that has never been done before.

The tools and technology needed for this project is the use of unity. Unity will be the software used to create the game graphics and the programming of the whole game. Along with needing this software the computer will need hardware that is viable use in game making. The hardware that would be needed is a core i7 processor accompanied with a Nividia GTX 1080. This hardware is needed as the computer needs to be able to run the game on its best possible graphics in order to see how smooth the gameplay is as well as how well the graphics have been designed.

The skills required to create a game of this scale would be someone who has a minimum of 3+ year's experience using programs such as unity and c++. The will also need to be able to do graphic design and debugging in order to prevent any errors from arising. In addition, the PC’s will need at least an i7 core processor and a Nividia GTX 1080 or equivalent. It is feasible too find the skills as many game programmers would be happy to have an opportunity to create their own bestselling game, along with this, this style of game does not require 10+ years of experience to create only imagination and 3+ years of experience in programming and graphic design. The cost of hardware that would be needed this being the PC is achievable as each part will cost roughly $1000 and as for the software unity and c++ are free so no cost is occurred there.

The outcome of creating a game of this type creates a whole new market of games it also gives other developers new ideas that they may not have considered and shows how creative a game can become.
